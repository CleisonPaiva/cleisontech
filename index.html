Toronto Infrastructure Risk Dashboard

Live Demo → Frontend → API → ETL →

Full-stack geospatial dashboard identifying which hospitals, schools, and subway stations in Toronto fall within TRCA flood-risk zones. Python ETL pipeline (GeoPandas, OSMnx, Shapely) processes open data from OpenStreetMap and the City of Toronto, calculates spatial risk using ST_Within operations, and loads results into Supabase/PostGIS. ASP.NET Core (.NET 10) API with Entity Framework and NetTopologySuite serves GeoJSON endpoints. React + TypeScript + ArcGIS Maps SDK frontend renders a choropleth map coloured by neighbourhood risk count, with UniqueValueRenderers for infrastructure layers, interactive risk ranking panel, and click-to-navigate between neighbourhoods.

Dashboard geoespacial full-stack que identifica quais hospitais, escolas e estações de metrô de Toronto estão em zonas de risco de inundação do TRCA. Pipeline ETL em Python (GeoPandas, OSMnx, Shapely) processa dados abertos do OpenStreetMap e da Cidade de Toronto, calcula risco espacial com operações ST_Within e carrega os resultados no Supabase/PostGIS. API em ASP.NET Core (.NET 10) com Entity Framework e NetTopologySuite serve endpoints GeoJSON. Frontend em React + TypeScript + ArcGIS Maps SDK renderiza choropleth por risco de bairro com painel de ranking interativo e navegação por clique nos bairros.

WellsGIS — Oil & Gas Well Platform

Live Demo → GitHub →

Full-stack geospatial platform for visualizing and managing 30,000+ oil & gas wells from ANP (Brazil's National Petroleum Agency). Built a bounding-box map engine that fetches only wells visible in the current viewport, zoom-adaptive point density, color-coded status markers, paginated searchable data table, CSV import with bulk upsert, and a slide-in detail panel. Backend in ASP.NET Core with Entity Framework and PostgreSQL. Frontend in React + TypeScript with ArcGIS Maps SDK.

Plataforma geoespacial full-stack para visualização e gerenciamento de mais de 30.000 poços de petróleo e gás da ANP. Motor de mapa com bounding box que carrega apenas poços visíveis no viewport, densidade adaptativa por zoom, marcadores coloridos por status, tabela paginada com busca, importação CSV com bulk upsert e painel lateral de detalhes. Backend em ASP.NET Core com Entity Framework e PostgreSQL. Frontend em React + TypeScript com ArcGIS Maps SDK.

React TypeScript ArcGIS SDK JS ASP.NET Core PostgreSQL Entity Framework PrimeReact Bounding Box API

Live Demo → GitHub →

Web map consuming Vitória city's public ArcGIS REST service. Dynamically lists all available layers, allows switching between them, and renders attribute-rich popup tables for each feature. Uses esriRequest to query the MapServer metadata at runtime.
