WellsGIS — Oil & Gas Well Platform

Live Demo → GitHub →

Full-stack geospatial platform for visualizing and managing 30,000+ oil & gas wells from ANP (Brazil's National Petroleum Agency). Built a bounding-box map engine that fetches only wells visible in the current viewport, zoom-adaptive point density, color-coded status markers, paginated searchable data table, CSV import with bulk upsert, and a slide-in detail panel. Backend in ASP.NET Core with Entity Framework and PostgreSQL. Frontend in React + TypeScript with ArcGIS Maps SDK.

Plataforma geoespacial full-stack para visualização e gerenciamento de mais de 30.000 poços de petróleo e gás da ANP. Motor de mapa com bounding box que carrega apenas poços visíveis no viewport, densidade adaptativa por zoom, marcadores coloridos por status, tabela paginada com busca, importação CSV com bulk upsert e painel lateral de detalhes. Backend em ASP.NET Core com Entity Framework e PostgreSQL. Frontend em React + TypeScript com ArcGIS Maps SDK.

React TypeScript ArcGIS SDK JS ASP.NET Core PostgreSQL Entity Framework PrimeReact Bounding Box API

Interactive GIS Viewer — Vitória

Web map consuming Vitória city's public ArcGIS REST service. Dynamically lists all available layers, allows switching between them, and renders attribute-rich popup tables for each feature. Uses esriRequest to query the MapServer metadata at runtime.

Mapa web que consome o serviço REST público do ArcGIS de Vitória. Lista dinamicamente todas as camadas, permite alternar entre elas e exibe tabelas de popup com todos os atributos. Usa esriRequest para consultar os metadados do MapServer em tempo real.

ArcGIS JS API FeatureLayer esriRequest REST API

Public Works Map — Vitória

Spatial visualization of public infrastructure projects in Vitória, ES. Features category filtering with definitionExpression, unique-value renderer with color-coded markers by project type, basemap switcher, and popup templates showing progress and delivery dates.

Visualização espacial de obras públicas em Vitória, ES. Inclui filtro por categoria com definitionExpression, renderer com marcadores coloridos por tipo de obra, seletor de basemap e popups com percentual de andamento e datas de entrega.

ArcGIS JS API FeatureLayer UniqueValueRenderer definitionExpression

3D Scene View — São Paulo

Interactive 3D urban visualization of São Paulo using ArcGIS WebScene and SceneView. Configures a custom camera position with heading and tilt, enables sun lighting with directional shadows, and allows full 3D navigation of the urban landscape.

Visualização urbana 3D interativa de São Paulo com ArcGIS WebScene e SceneView. Configura posição de câmera customizada com heading e tilt, habilita iluminação solar com sombras direcionais e permite navegação 3D completa do ambiente urbano.

ArcGIS SceneView WebScene 3D Camera Sun Lighting
